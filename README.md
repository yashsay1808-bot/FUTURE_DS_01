 FUTURE_DS_01 - Business Sales Performance Analysis

 📊 Objective
 
Analyze business sales data to identify trends, top products, and regional performance.

 🛠 Tools Used
- Excel / Power BI
  
 🔍 Key Insights
- Electronics is the top-performing category
- Laptops and phones generate highest revenue
- North and South regions perform best
  
 📈 Features
- Revenue trend analysis
- Product performance visualization
- Regional sales insights
  
 💡 Conclusion
 
This analysis helps businesses make data-driven decisions to improve sales and profitability.
